# PoetryEaseWrite

A clean, modern web app designed to make writing and editing poetry easier — whether you're drafting on a computer or making quick tweaks.

## Features

- Distraction-free writing interface
- Real-time formatting and preview
- Easy export to Markdown, TXT, or PDF
- Theme switching (light/dark)
- Keyboard shortcuts for power users

## Live Demo

https://easepoet.drew-gnr.xyz/

## Tech Stack

- HTML, CSS, JavaScript (vanilla)
- No backend required (fully client-side)

## Installation (Self-hosted)

Just open `index.html` in any browser or host it statically.
