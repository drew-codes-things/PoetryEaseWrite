# PoetryEaseWrite

Modern, distraction-free web app for writing and editing poetry with real-time preview and export options.

## Technical Stack

- Pure vanilla HTML + CSS + JavaScript (fully client-side)
- No backend required
- Theme switching (light/dark)
- Keyboard shortcuts
- Export to Markdown / TXT / PDF
